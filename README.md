## 使用技術

  ・Vue.js 2.6.14
  
  ・Nuxt.js 2.15.8
  
  ・Vuetify 1.12.3
  
  ・Firebase 9.7.0
  
  ・NewsAPI

### 機能一覧
・ユーザー新規登録

・ログイン（Google・匿名ログイン）

・海外の記事を取得する用にNewsAPIを使用

・詳細記事のフォントサイズやテキストの配置・下線追加・太字変更などの設定可能

・ブックマーク機能

・単語登録機能

・理解度テスト機能
